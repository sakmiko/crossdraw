## 4. 准确性纪律（已落地）

1. 配时轴 = **C**；主相位 G+Y+AR 闭合检查  
2. 放行矩阵 / 流量图 / 分析汇总 **同源层**  
3. 绿波带宽 **圆环弧相交**，禁止装饰缩水  
4. 展宽加宽 = count×width，**禁止 0.35 系数**  
5. 冲突矩阵与 `detectPhaseConflicts` 同规则  

## 5. 不足与风险

1. **App.tsx 体量**：模式面板继续膨胀，宜按 mode 拆文件  
2. **冲突模型**：方位启发式，非完整冲突点坐标几何  
3. **绿波多走廊**：独立优化/导出齐了，但无走廊拓扑地图  
4. **测试**：E2E 每轮单场景；宜恢复冒烟套件多页  
5. **预览环境**：:4175 隧道易 502，需部署文档化
