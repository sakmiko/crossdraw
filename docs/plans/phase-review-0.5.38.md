## 4. 不足与风险

1. **App.tsx** 仍大（流量/渠化/绿波/分析内联）— 本轮拆流量  
2. 行人模型为「与机动车相位并行」，非完整 NEMA 行人相位  
3. 冲突几何为示意路径，非轨迹仿真  
4. E2E 仍单场景轮换；宜加冒烟多页  
5. 预览隧道稳定性
