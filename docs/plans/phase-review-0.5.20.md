# Crossdraw 阶段复盘与路线图（v0.5.20 基线）

**日期：** 2026-07-16  
**当前版本：** 0.5.20  
**仓库：** sakmiko/crossdraw  

## 1. 目标（不变）

- 本地优先、开源交叉口出图与配时/绿波/分析工具  
- 专业图表：矢量优先、与数据热联动、依据国标/论文  
- 层次渐进：工程 → 方案（渠化/流量/信号）→ 模式编辑 → 图/表/导出  
- 版本节奏：**每次 +0.0.1**，小步可测可推送  
- 竞品对标：RoadGee 公开能力 + 教材图式（非闭源复刻）

## 2. 已完成（相对 RoadGee 公开树）

| 模块 | 状态 | 说明 |
|------|------|------|
| 渠化几何示意 | 中 | 十字/T/Y/斜交/环；图例/图框/比例尺有；非测绘级 |
| 流量 | 中高 | 自然/高峰同源、流向图、柱图 |
| 信号配时 | 中高 | 多方法+固定周期；表图对齐；环栏轴=C |
| 放行矩阵 | 高 | L/T/R 逐格同源 |
| 运行评价 | 中 | HCM 风格延误/LOS；同源徽章；拼图导出 |
| 方案比选 | 中 | 独立比选模式 + 并排配时 |
| 绿波 | 中 | 多方法、锁定 o、圆环弧带宽、交互时距图 |
| 断面 | 中 | 标准横断面矢量 |
| 导出 | 中高 | 导出中心 + 同源门禁 |
| 地图 | 低 | OSM 示意底图 |
| Vissim | 低 | CSV 中间表，非完整 .inpx |
| UI | 中高 | 双主题、方案树层次、检查器层次条 |

## 3. 主要不足

1. **渠化**：角部/展宽/安全岛/可变车道/辅路等仍粗；图注密度与 CAD 出图习惯有差距  
2. **绿波**：非完整 MAXBAND-MIP；多走廊/地图选点未做  
3. **分析**：未做转向当量细表、行人/非机影响  
4. **图表**：部分图例未统一注册；打印专用布局弱  
5. **工程化**：App.tsx 过大；图表模块需 registry 便于维护  
6. **RoadGee 缺口**：完整 inpx、云项目、地图设计模式、复杂搭接 UI

## 4. 近程迭代计划（+0.0.1）

| 版本 | 主题 |
|------|------|
| 0.5.21 | 渠化图框/图例/指北针专业度 + 进口标注规范 |
| 0.5.22 | LOS 色带与延误阈值同源图例 + 分析表列对齐 |
| 0.5.23 | `chartRegistry`：图种清单/数据源/导出 id 映射 |
| 0.5.24 | 打印/导出预览页（A4 示意拼版） |
| 0.5.25+ | 渠化安全岛参数、绿波多走廊骨架、inpx 深化… |

**原则：** 每版单主题、可测、可回滚；专业数字只从 domain 同源层出图。
