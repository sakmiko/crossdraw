# Crossdraw 阶段复盘 v0.5.65（弱项深化 + 专业呈现）

**版本：** 0.5.65  
**主题：** 把明确仍弱的能力做深，并保证界面/图表/报告专业美观。

## 本版交付

| 模块 | 内容 | 诚实边界 |
|------|------|----------|
| VISSIM | `.inpx.xml` 开放交换 + CSV + summary | 非 PTV 私有二进制 |
| 绿波 | MAXBAND 离散搜索 + 锁定 | 非商业 MIP 求解器 |
| 走廊 | lat/lon 选点 + 专业地图 SVG | 示意非测绘底图 |
| 辅路 | 参数 + 几何带 | 侧辅示意 |
| 行人 | 专用 exclusive 相位 | 非完整 NEMA 相位机 |
| 断面 | 滑条改宽回写渠化 | 非 CAD 拖拽控制点 |
| 报告 | 5 页 HTML 打印/PDF | 浏览器打印路径 |

## 视觉

- multimodal / 走廊图：圆角面板、图例、网格、数值标注  
- 断面：交互色带高亮 + 滑条  
- 报告：A4 分页、KPI 卡、表头层级  

## 测试

单测 148+；E2E 覆盖壳层/渠化辅路/流量图/行人相位/断面/分析导出/绿波 MAXBAND/比选。
