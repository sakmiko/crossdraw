# UI 设计规范（摘要）

**状态：** Accepted  
**版本：** 0.1.0

## 1. 原则

1. 一屏一目标  
2. 80/20 渐进披露  
3. 智能默认（国标）  
4. 空间一致性  
5. **禁止 emoji**  

## 2. 布局尺寸（初始 token）

| Token | 值 |
|-------|-----|
| titleBarHeight | 40px |
| leftSidebarWidth | 260px |
| rightSidebarWidth | 320px |
| statusBarHeight | 28px |
| panelSectionGap | 12px |
| controlHeight | 32px |

## 3. 色彩

- 中性灰：框架  
- 主色蓝：强调/选中  
- 成功绿 / 警告黄 / 错误红：合规  
- 画布背景：浅灰或图纸白（可配置）  

## 4. 字体

- UI：系统字体栈  
- 图纸标注：可嵌入开源字体（SIL/OFL），需许可证收录  

## 5. 模式切换

右侧顶部 Segmented control：渠化 | 流量 | 信号 | 断面  
不改变顶栏全局按钮位置。

## 6. 密度

专业工具中等密度；表格行高紧凑；禁止营销大留白。
