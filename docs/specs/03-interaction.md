# 交互规格

**状态：** Accepted  
**版本：** 0.1.0

## 1. 四手势

| 手势 | 含义 |
|------|------|
| 单击 | 选择 / 激活 |
| 拖拽 | 平移画布 / 调整句柄（V1 句柄可少） |
| 双击 | 重命名 / 聚焦编辑 |
| 右键 | 上下文菜单 |

## 2. 画布

| 操作 | 行为 |
|------|------|
| 中键拖 / Space+左键 | 平移 |
| 滚轮 | 缩放至光标 |
| 左键点选 | 选中进口/相位 |
| Esc | 清空选择 |

## 3. 面板

- Tab 在流量表内切换单元格  
- Enter 确认，Esc 取消编辑  
- 非法数字：即时 WARN，不写坏数据  

## 4. 命令面板

Ctrl+K：搜索命令与对象（V1 可先命令列表）。

## 5. 危险操作

删除方案 / 删除进口：Modal 确认。
