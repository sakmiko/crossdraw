# 默认值与规范引用

**状态：** Accepted  
**版本：** 0.1.0

## 1. 原则

默认值 = 国标/行标**推荐值**；超出给 WARN，不强制 BLOCK（除非物理不可能，如宽度≤0）。

## 2. V1 默认表（摘录）

| 参数 | 默认 | 参考 |
|------|------|------|
| 机动车道宽 | 3.5 m（主干）/ 3.25 m（次干可选模板） | CJJ 37-2012 表5.3.2 思路 |
| 人行道宽 | 3.0 m | CJJ 37-2012 表5.3.4 思路 |
| 交叉口展宽段长 | 70 m | CJJ 152-2010 4.2.13 思路 |
| 右转渠化半径 | 15 m | 工程常用，可标「经验默认」 |
| 转角半径 | 8 m | 经验默认 |
| PHF | 0.95 | 常用 |
| 黄灯 | 3 s | 常用 |
| 启动损失 | 3 s | 与公开工具口径对齐 |
| 饱和流率 | 1650–1800 pcu/h/ln 可配 | 借道左转文档口径 1650 |
| 车长换算排队 | 5.5 m/veh | 公开工具口径 |

## 3. 合规级别

| 级 | 含义 |
|----|------|
| OK | 落在推荐区间 |
| WARN | 可用但偏离推荐 |
| BLOCK | 无法计算/几何自交/除零 |
