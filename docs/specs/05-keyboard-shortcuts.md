# 键盘快捷键

**状态：** Accepted  
**版本：** 0.1.0

| 快捷键 | 动作 |
|--------|------|
| Ctrl+N | 新建 |
| Ctrl+O | 打开 |
| Ctrl+S | 保存 |
| Ctrl+Shift+S | 另存 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z / Ctrl+Y | 重做 |
| Ctrl+E | 导出菜单 |
| Ctrl+K | 命令面板 |
| Delete | 删除选中（需确认的要确认） |
| Esc | 取消/清选择 |
| 1/2/3/4 | 渠化/流量/信号/断面模式 |
| Space+拖 | 平移 |
| Ctrl+0 | 缩放到适合 |
| Ctrl+= / - | 缩放 |

平台：macOS 上将 Ctrl 显示为 Cmd（实现映射）。
