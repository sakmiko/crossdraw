# D08 — 方案树

**依赖：** D05  
**产出：** 左栏方案树完整行为  

## 1. 目标

对齐 RoadGee 心智：渠化 ⊃ 流量 ⊃ 信号；复制深拷贝；突破 3 上限。

## 2. 行为

| 操作 | 语义 |
|------|------|
| 新增渠化 | 空或从模板 |
| 复制渠化 | 含全部子流量/信号，新 ID |
| 新增流量 | 挂当前渠化 |
| 复制流量 | 含子信号 |
| 新增信号 | 挂当前流量 |
| 删除 | 确认；级联 |
| 激活 | 更新 active 指针并刷新画布 |

默认 `maxSchemes=10` 可配置。

## 3. 通过条件

| ID | 条件 |
|----|------|
| UT-TREE-001 | 复制后 id 全不同 |
| UT-TREE-002 | 删除渠化后子节点清空 |
| UT-TREE-003 | 可建第 4 个流量（相对 SaaS 3 限） |
| IT-TREE-001 | UI 操作反映到 store |
