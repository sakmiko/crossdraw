# 搭接相位

搭接相位也称早启迟断式信号控制，常用于交通流量不对称的交叉口。

详细示例《 [例2：十字交叉口-早启迟断式信号控制（搭接相位）](https://www.roadgee.com/doc/example2)》。

如下图，西进口左转车流量远大于东进口左转车流量，需要增加一个相位进行过渡（第2相位为搭接相位，也称叠加相位）。
