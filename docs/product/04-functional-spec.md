## 2. 模式规格

### 2.1 渠化模式

**画布：** 交叉口平面渠化图（车道、停止线、人行、岛、箭头、展宽）。  
**面板分组：**

| 分组 | 字段（V1） |
|------|------------|
| 道路 | 名称、方向索引、设计速度 |
| 进口 | 车道数、车道宽度或宽度数组、展宽数、展宽宽、展宽段长、外侧渐变、内侧偏移、内侧渐变 |
| 出口 | 车道数、宽度、展宽相关 |
| 右转渠化 | 启用、固体/划线、单独入口/出口、宽度、曲度/半径 |
| 中分 | 形式枚举、带宽 |
| 行人/非机 | 人行宽度、非机启用、分隔形式、宽度 |
| 显示 | 指北针、背景色、高亮当前进口 |

**交互：** 选中进口 → 面板绑定；改参 → `rebuildIntersection(schemeId)`。

### 2.2 流量模式

**画布：** 流率箭头图叠在渠化底图上。  
**面板：**

| 字段 | 规则 |
|------|------|
| 转向矩阵 | 各进口 ×（掉头/左转/直行/右转）自然车辆数 |
| 大车比例 | 0–1，0 表示不折算 |
| PHF | 默认 0.95，1 表示不折高峰 |
| 饱和流率 | 分车道或车道组 |
| 绘图属性 | 颜色方案、宽度映射、字号 |

**折算：** 先按大车比得 PCU，再按 PHF 得高峰小时流量（细节见分析规格）。

### 2.3 信号模式

**画布：** Ring-Barrier 或相位条带图 + 相位箭头示意。  
**面板：**

| 字段 | V1 |
|------|-----|
| 周期 C | 秒 |
| 相位列表 | 名、绿、黄、全红、放行转向集合 |
| 顺序 | 上移下移/拖拽 |
| 冲突 | 基础冲突矩阵提示 WARN |
| 自动配时 | V1 显示「即将推出」禁用或只读说明 |

### 2.4 横断面模式

**入口：** 渠化图各进口「生成断面」或模式内选择进口。  
**画布：** 该进口标准横断面简绘。  
**组件（V1）：** 人行、非机、机动车道、中分、路缘带（可配置显隐）。  
**联动：** 车道数/宽来自渠化；改渠化后断面标记 stale 并支持一键刷新。

## 3. 方案树规则

```
Project
 └─ ChannelizationScheme[]   // 默认上限 10，可配置
      ├─ geometry + channel params
      └─ FlowScheme[]
           ├─ volumes + sat flow + style
           └─ SignalScheme[]
                └─ phases + cycle
```

| 操作 | 语义 |
|------|------|
| 复制渠化 | 深拷贝几何与全部子流量/信号 |
| 复制流量 | 深拷贝流量字段与子信号 |
| 复制信号 | 仅信号 |
| 删除 | 子节点级联删除；需确认 |

## 4. 导出规格

| 格式 | V1 要求 |
|------|---------|
| PNG | 当前模式视口；可选 1×/2× |
| SVG | 矢量，文本可选转曲 |
| DXF | 图层：ROAD / MARKING / ISLAND / ANNO / FRAME |
| RTP | 完整项目 |

## 5. 文件与会话

- 格式：`specs/02-file-format.md`  
- 自动保存：每 30s 写 `.autosave` 旁路文件  
- 启动时检测未恢复会话  

## 6. 错误与空状态

| 状态 | 表现 |
|------|------|
| 无道路 | 空画布引导「添加进口/使用模板」 |
| 流量全 0 | 箭头隐藏，分析 WARN |
| 相位绿全 0 | BLOCK 提示无法配时 |
| 文件损坏 | 报错 + 尝试备份 |

## 7. V1.1+ 功能锚点（规格占位）

- 分析四图：热力/表；公式见 `specs/04-analysis-algorithms.md`  
- Webster：入口参数 VC、PHF、启动损失、固定周期开关  
- 绿波：独立工作区，不阻塞交叉口 V1  

## 8. 验收场景（摘要）

1. 模板「标准十字」一键创建  
2. 东进口车道 2→3，渠化与断面更新  
3. 输入流量出现箭头  
4. 两相位信号可编辑周期  
5. 导出三份文件可打开  
6. Kill 进程后恢复  

详细测试门禁见 `plans/fixed-testing-gates.md`。
