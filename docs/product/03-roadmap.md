# Crossdraw 版本路线图与范围边界

**状态：** Accepted  
**版本：** 0.1.0

## 1. 版本哲学

```
V1    生成闭环可用（出图、保存、导出）
V1.1  分析与兼容硬化
V1.2  自动配时与方案对比
V2    绿波 / 地图 / 实景断面 / 深度仿真导出
V3    插件生态与路网级能力（谨慎）
```

**范围冲突时：路线图 > 功能愿望清单。**

## 2. V1.0 — 单交叉口生成器（MVP+）

### 2.1 必须交付

- 项目：新建、打开、保存、自动保存、崩溃恢复  
- 方案树：渠化 / 流量 / 信号（复制、删除、重命名）  
- 几何：十字路口稳定生成；T 型尽力  
- 渠化参数：车道、宽度、展宽、渐变、右转渠化、中分、人行、非机（基础）  
- 流量：转向矩阵、大车比、PHF、饱和流率、箭头图  
- 信号：手动相位、周期、Ring-Barrier、冲突基础提示  
- 横断面：简绘 + 从进口一键生成  
- 导出：PNG、SVG、DXF（基础图层）  
- 规范：默认值 + 基础 WARN  
- 示例：≥ 3 个 `.rtp` 样例  

### 2.2 V1 明确不做

| 项 | 去向 |
|----|------|
| Webster 自动配时（可放禁用入口） | V1.2 |
| 饱和度/延误/排队/LOS 计算 | V1.1 |
| 多方案对比仪表盘 | V1.2 |
| 借道左转完整语义 | V1.1 |
| 辅路完整 | V1.1 |
| 可变车道工作流 | V1.2 |
| 干道绿波 | V2 |
| 地图底图 | V2 |
| 横断面实景 | V2 |
| Vissim .inpx | V2（V1.1 可 CSV） |
| 多交叉口协调 | V3+ |
| 实时协作 | 不做（除非独立项目） |

### 2.3 V1 成功标准

用户路径 100% 可测：

```
新建 → 生成十字 → 改展宽 → 输入流量 → 编辑相位 → 生成断面 → 保存 rtp → 导出 PNG/SVG/DXF
```

## 3. V1.1 — 分析与硬化

- 饱和度、延误、排队、LOS（公式见 `specs/04-analysis-algorithms.md`）  
- Excel 详/简导出  
- Vissim CSV  
- 红灯右转、借道左转、倾斜/转角、搭接相位基础  
- DXF 图层与标注增强  
- 完整度更高的国标规则库  
- 一键修复（车道不一致等）  

## 4. V1.2 — 智能与对比

- Webster 自动配时  
- 多方案对比（饱和度/延误/排队）  
- 可变车道多方案工作流  
- 深色主题、命令面板完善  
- 性能：复杂相位下的编辑流畅度  

## 5. V2 — 扩展产品线

- 干道绿波：经典数解、优化数解，再生 MaxBand/图解  
- 地图选点与坐标  
- 横断面实景组件库  
- Vissim .inpx（能力范围内）  
- 更多交叉口形态模板  

## 6. 差异化「更强」清单（相对典型 SaaS）

| 能力 | 说明 |
|------|------|
| 无限本地保存 | 无次数、无水印 |
| 方案数可配置 | 默认高于 3×3×3 |
| `.rtp` 可版本管理 | 适合 Git |
| 引擎开源可测 | 公式与几何可审计 |
| 规范溯源 | 默认值带来源条文 |
| 导出开放 | DXF/CSV 优先保证「能用」 |
